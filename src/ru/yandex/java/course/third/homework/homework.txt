3.Приложение для продажи товаров в магазине (к примеру магазин одежды).
На старте у магазина есть перечень доступных товаров с указанным количеством.
Покупка состоит из нескольких стадий:
Нополнение корзины покупателя товарами.
Оплата покупки на кассе.
Печать чека. Для каждого товара указана его стоимость, величина скидки и итоговые значения по стоимости и скидке для всего чека.
У покупателя есть возможность выбрать способ оплаты: наличными или по карте + у покупателя могут быть дисконтные карты с разной величиной скидки.
В ходе разработки программы надо соблюдать принципы SOLID
